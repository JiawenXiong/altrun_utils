# Altrun Utils [English](README_EN.md)

Altrun 快捷方式配置文件合并工具，用于合并和压缩 Altrun 的快捷方式列表配置。

## 功能特性

- **单文件去重**：对单个配置文件进行去重压缩
- **配置文件合并**：合并两个 Altrun 快捷方式配置文件
- **冲突处理**：自动处理重名快捷方式的冲突
- **重复压缩**：合并相同路径的快捷方式条目，智能合并关键词
- **自动备份**：合并后自动备份输入文件（.ori1/.ori2 后缀）
- **GB2312 编码支持**：正确读写 GB2312 编码的配置文件

## 使用方法

```bash
python altrunMerger.py <配置文件1> [配置文件2]
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `配置文件1` | 第一个配置文件路径（必填） |
| `配置文件2` | 第二个配置文件路径（可选） |

输出文件固定为 `ShortCutList.txt`

### 示例

**单文件去重**
```bash
python altrunMerger.py ShortCutList.txt
```

**两文件合并**
```bash
python altrunMerger.py ShortCutList_local.txt ShortCutList_hw.txt
```

## 工作原理

1. **读取配置**：解析 GB2312 编码的配置文件
2. **按键合并**：以第 3 字段（快捷方式名称）为键进行合并
3. **冲突处理**：名称冲突时自动添加后缀 `_0`、`_1` 等
4. **路径压缩**：相同路径（第 5 字段）的条目合并，关键词智能拼接
5. **自动备份**：输入文件重命名为 `.ori1`、`.ori2` 后缀保存

## 配置文件格式

配置文件使用 `|` 分隔字段，示例：

```
字段1 | 字段2 | 快捷方式名称 | 关键词 | 路径 | ...
```
